# Python食堂外卖系统 python397

## 项目概述

本项目是一个基于Python的食堂外卖系统，包含源码、数据库以及开发文档，旨在为用户提供便捷的下单与配送服务。

## 技术栈

- **后端语言**: Python
- **框架**: Django
- **数据库**: Mysql
- **架构模式**: B/S

## 功能模块

### 前台功能

- 首页
- 菜品信息
- 新品资讯
- 个人中心
- 购物车

### 管理员功能

- 首页
- 商家管理
- 用户管理
- 商品类型分类
- 商品信息管理
- 系统管理
- 个人中心

## 系统角色

- **普通用户**: 可以下单、查看菜品信息等。
- **管理员**: 负责商家管理、用户管理、商品管理以及系统管理。

## 运行环境

- Python 3.x
- Django 2.x
- Mysql 5.x
- 适用于主流浏览器

## 部署步骤

1. 配置Python环境
2. 安装依赖库
3. 配置Mysql数据库
4. 运行系统

```
项目根目录/

```
